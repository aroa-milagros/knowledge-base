# Nmap

## Purpose

Network discovery and security auditing.

## Common Commands

```bash
nmap 192.168.1.1
nmap -sV 192.168.1.1
nmap -O 192.168.1.1
nmap -A 192.168.1.1
```

## Useful Flags

| Flag | Purpose |
|---------|---------|
| -sS | SYN Scan |
| -sV | Version Detection |
| -O | OS Detection |
| -A | Aggressive Scan |
