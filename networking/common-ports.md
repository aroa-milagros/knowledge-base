# Common Ports

| Port | Protocol | Service |
|--------|----------|----------|
| 20/21 | TCP | FTP |
| 22 | TCP | SSH |
| 23 | TCP | Telnet |
| 25 | TCP | SMTP |
| 53 | TCP/UDP | DNS |
| 80 | TCP | HTTP |
| 443 | TCP | HTTPS |

## Notes

- SSH is the preferred secure alternative to Telnet.
- DNS commonly uses UDP but may use TCP for zone transfers.
